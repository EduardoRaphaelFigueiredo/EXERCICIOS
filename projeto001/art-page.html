Artefatos para Yanfei DPS

Yanfei é uma carregadora (DPS) pyro muito efetiva, que usa muitos ataques carregados por ser uma usuária de catalisador.
Sendo assim, seus sets seguem um padrão de areias com porcentagem de ataque, taça com foco em bônus de dano pyro e porcentagem de taxa critica ou dano critico na coroa, o que for mais necessário na sua build - seguindo sempre a razão de 1:2 ao se tratar de taxa:dano.
Além disso, nos substatus é interessante buscar por %atk e porcentagem de recarga de energia, visto que Yanfei tem alguns problemas nessa área devido o alto custo no seu burst, que é vital para seu DPS.

Set Bruxa das chamas carmesim

O set da Bruxa das chamas carmesim é ótimo em times com Xingqiu, Kaeya, Fischl ou qualquer outro personagem que crie reações de dano como Vaporizar, Derreter ou Sobrecarga.
O set tem os seguintes efeitos: 2 peças dão 15% de bônus de dano pyro, o que aumenta o dano geral de qualquer DPS pyro. Já nas 4 peças, o set concede um bônus de 40% de dano nas reações de Sobrecarga e Queimada (dendro + pyro), e de 15% em Vaporizar e Derreter. Além disso, ele também concede até 3 stacks de dano pyro bônus conforme a habilidade elemental é usada.
Esse set é uma boa opção em times com variação de elementos entre os personagens, como Rosaria, Mona, Barbara e os já citados anteriormente. Em casos de time mono pyro, ou com 2+ personagen anemo/geo, esse set não é tão ideal, já que uma boa parte de seu efeito de 4 peças é perdido.

Set Trupe Itinerante

O set da Trupe itinerante é ótimo em contexto geral, já que não precisa de reações ou situações especificas para funcionar.
O set tem os seguintes efeitos: 2 peças dão 80 de maestria elemental, o que aumenta o dano de reações. Já nas 4 peças, o set aumenta em 35% o dano dos ataques carregados de personagens que usam catalisador e arco, se aplicando para a Yanfei.
Esse set é uma boa opção em todos os times da Yanfei, já que ela usa o mesmo sistema de 3-4 ataques normais e um carregado.

Set Reminiscência nostalgica

O set da Reminiscência nostalgica é poderoso porém arriscado, já que já que possui um efeito forte ao custo de recursos importantes.
O set tem os seguintes efeitos: 2 peças dão 18% de ataque, o que aumenta o dano geral. Já nas 4 peças, o set aumenta em 50% o dano de todos os ataques do personagem, ao custo de 15 de energia elemental (a que usamos para usar a explosão elemental).
Esse set é uma boa opção em times com alguma bateria, como o time de Yanfei Sobrecarga, com a Raiden Shogun como bateria para Yanfei, já que seu burst é essencial para seu DPS.
